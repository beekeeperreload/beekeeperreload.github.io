Problem : Akun sering logout sendiri.

Jika anda mengalami kendala dimana akun sering keluar (logout) sendiri, silahkan dicoba update aplikasi ke versi terbaru

1. Cari di Playstore aplikasi BeeKeeper Reload atau bisa klik link ini

2. Klik update dan tunggu sampai selesai.


3. Jika muncul eror ketika install seperti gambar dibawah, silahkan di uninstall / hapus terlebih dahulu aplikasi yang lama.


4. Versi terbaru yaitu 3.0.5, untuk cek nya bisa klik garis tiga di kiri atas


5. Dan tulisan versi yang terinstall ada di kiri bawah
